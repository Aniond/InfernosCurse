# Stylized Grass Shader for Unity 6
*by Staggart Creations*

## Project Requirements
Compatible with Unity 6000.0.22f1+ and a Universal Render Pipeline setup.

## Usage instructions
Navigate to the Package/Stylized Grass Shader/Prefabs folder and drag any one of them into the scene

## Complete documentation
Go to *Help/Stylized Grass* to open the asset window. Links to resources are available there.
